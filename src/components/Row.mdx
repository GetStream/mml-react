# Row

It helps display content in a grid. Typically used with [`Col`](/components/col).

## Properties

<Props of={Row} />

## MML string

```xml
<mml>
  <row>
    <col width="4"><md>**Title**</md></col>
    <col width="8"><md>*Book title*</md></col>
  </row>
  <row>
    <col width="4"><md>**Year**</md></col>
    <col width="8"><text>1988</text></col>
  </row>
</mml>
```
