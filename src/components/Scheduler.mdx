# Scheduler

Datetime picker with availability using [iCal](https://icalendar.org/).

## Properties

<Props of={Scheduler} />

## MML string

```xml
<mml name="massage">
  <text>When would you like to schedule your massage?</text>
	<scheduler name="appointment" duration="30" interval="30" selected="2020-11-16T10:30:00.000Z"/>
  <button name="action" value="reserve">Reserve</button>
</mml>
```

## Basic usage

<Playground>
