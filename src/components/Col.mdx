# Col

> This component is not exposed as `mml` string, it is used internally in other mml components and it can be imported to create your custom ones.

It helps display content in a grid. Typically used with [`Row`](/components/row)

## Properties

<Props of={Col} />

## MML string

```xml
<mml>
  <row>
    <col width="4"><md text="**Title**"/></col>
    <col width="8"><md text="*Book title*"/></col>
  </row>
  <row>
    <col width="4"><md text="**Year**"/></col>
    <col width="8"><text>1988</text></col>
  </row>
</mml>
```
