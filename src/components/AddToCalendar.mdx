# AddToCalendar

It allows to define an event that a user can quickly add to her/his own calendar, we support Google, Apple and Outlook calendars.

## Properties

<Props of={AddToCalendar} />

## MML string

```xml
<mml>
  <text>The party is going to happen!</text>
  <add_to_calendar title="Christmas party" start="2020-12-24T18:00:00.148Z" end="2020-12-26T15:00:00.148Z" description="The gift is your presence" location="Boulder, CO" />
</mml>
```

## Usage


### Basic

<Playground>
  <Text text="Your massage appointment has been confirmed!" />
  <AddToCalendar
    title="Massage with Julia"
    start={new Date(2021, 1, 1, 10, 0, 0, 0)}
    end={new Date(2021, 1, 1, 11, 0, 0, 0)}
    location="Boulder, CO"
    description="Please arrive 15 minutes early"
  />
</Playground>

## With date strings

<Playground>
  <Text text="Your massage appointment has been confirmed!" />
  <AddToCalendar
    title="Massage with Julia"
    start="01/02/2021 10:00:00"
    end="01/02/2021 11:00:00"
    location="Boulder, CO"
    description="Please arrive 15 minutes early"
  />
</Playground>
