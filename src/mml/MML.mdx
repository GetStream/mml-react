# MML

This component accepts a mml string and render it as react components.

If there are any stateful tag, it will gather all the inital states and and expose them in [`MMLContext`](/mml-context).

It also accepts custom convertor which allows you to exapand the default tags and add more compoennts.

## Basic usage

<Playground>
  <MML source="<mml>hi</mml>" />
</Playground>

## Basic usage with stateful components

<Playground>
  <MML
    source={`
      <mml name="donut_form">
        <text>How many donuts?</text>
        <number name="donuts" value="3" />
        <button>Confirm</button>
      </mml>
    `}
  />
</Playground>

## Callback support

If onSubmit callback return a promise, it will be awaited.

<Playground>
  <MML
    onSubmit={(data) => console.info(data)}
    source={`
      <mml>
        <input name="answer" value="42" />
        <button>Confirm</button>
      </mml>
    `}
  />
</Playground>

## Invalid MML string

In case of an error, you can override default Error component

<Playground>
  <MML source="<mml><invalid /></mml>" Error={({ error }) => <div style={{ color: 'red' }}>{error}</div>} />
</Playground>

## With `card` type

Shows the mml content inside a card like UI, see the [`Number` component usage](/components/number) as a further example.

<Playground>
  <MML
    source={`
      <mml type="card">
        <input name="answer" value="42" />
        <button>Confirm</button>
      </mml>
    `}
  />
</Playground>
